アプリ概要

情報Ⅰ 快速ドリルは、情報社会と問題解決、デジタル表現、コンピュータの仕組み、 ネットワーク、セキュリティ、プログラミング、データの活用などの分野を学習できるアプリです。

問題演習、分野別演習、復習、学習履歴の確認ができます。

広告は表示せず、行動追跡を目的とした解析は行いません。学習履歴や設定情報は主として端末内に保存され、外部サーバーへの送信は、お問い合わせ時や各プラットフォームによる購入処理等の限定的な場合を除き、想定していません。

主な機能

おまかせ演習

学習状況に応じて、確認したい問題を出題します。

分野別演習

情報Ⅰの各分野から、重点的に学びたい範囲を選んで演習できます。

解き直し

間違えた問題や自信のない問題を、復習対象として確認できます。
